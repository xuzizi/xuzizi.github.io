宕机138天

发表于 2017-02-11 更新于 2023-05-22 分类于 杂记 本文字数： 829 阅读时长 ≈ 1 分钟

[hermit auto=”1” loop=”0” unexpand=”0” fullheight=”0”]remote#:2[/hermit]  

正月十五，过年结束。

在家里伏案一天，终于把网站又弄了起来。

删掉了大量的资源文章，然后使用了最简单的主题。

看起来low low的。

________________________________

要不是昨日新网客服电话推销，我都不会记得我还有个网站了。

再次搬到阿里云，代码上传完立马就收到监控邮件提醒。

138天。

________________________________

去年开学之后，就跟各位同学全身心的投入到找工作的大军之中了。

新网打电话说网站太占内存要升级，完全没在care，

我说暂时不会升级更贵的服务器，

我看运行的挺好的，你总不能给我关了吧。

…

然后特么的就给我关了。

________________________________

当然这都是小事，当时在写简历的我没时间扯淡，

倒是工作的事才让人糟心。

在武汉和北京徘徊了好久，尽管大家都觉得回武汉比较好，可是我还是固执的要留在北京。

有的事情就是道理都懂，可是我不试试我不甘心。

________________________________

朋友聚会上了解到大家的生活都很充实。

在家乡可以有足够的工资和闲适的生活，

北上广深工作的同学能有很好的待遇和发展空间，

在武汉可以对收支有一个很好的平衡。

再看我，感觉是哪一头都不占。

我可能我会喜欢那种熬夜加班奋斗的生活吧。

在完成『稳定的工作』『幸福的婚姻』之前，

让我一个人安安静静的做个奋斗者吧。

________________________________

在我当机138天后的新学期，最后一个学期，正式工作前的5个月。

我也定几个小目标好了，我希望可以好好写论文，锻炼健身，然后继续钻研一些有用的东西。电器维修就算了，昨天电池没炸就感激涕零了。😅

bug什么的，慢慢调吧，我有时间。

WTF，该网站因为过度消耗资源暂时无法访问？！
