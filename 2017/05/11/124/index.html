去掉word中尾注上面的横线和空行

发表于 2017-05-12 更新于 2023-05-22 分类于 杂记 本文字数： 151 阅读时长 ≈ 1 分钟

以Word 2010为例，步骤如下： （1）插入尾注后，将视图切换为“草稿”。 （2）将选项卡切换到“引用”，点击“显示备注”。 （3）然后在下方的下来列表里选择“尾注分隔符”。 （4）按键盘上的Backspace删除键直接删除横线，然后右键：段落——行距：固定值，0.7磅【不设置的话那个地方会多出一个空行】。
